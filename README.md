# Promo Video Maker

懒人项目推广视频生成器：上传产品截图，为每张截图填写讲解文稿，生成配音和字幕后导出 MP4。

## 当前 MVP

- 添加多张截图
- 每张截图填写讲解文稿
- 每张截图手动设置展示时长
- 根据文稿估算展示时长
- 使用 macOS `say` 生成本地配音
- 自动生成字幕
- 支持 B 站横版 `16:9` 和抖音竖版 `9:16`
- 使用 `ffmpeg-static` 合成 MP4

## 不做的内容

第一版不做复杂剪辑能力：

- 不做镜头运动
- 不做鼠标模拟
- 不做复杂转场
- 不做多轨剪辑
- 不做出镜录制

## 本地运行

```bash
npm install
npm run dev
```

## 构建验证

```bash
npm run build
```
